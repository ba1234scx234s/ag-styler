# AG Styler

A Chrome extension for [autogate.co](https://autogate.co) that lets you swap fonts and colours across the entire app.

- **Font swap** — replaces Open Sans with Manrope, Inter, Atkinson site-wide
- **Colour swap** — replace any hex colour with another (supports multiple swaps)
- **Radius swap** — replace the button radii

---

## Installation

Chrome extensions can be loaded directly from your computer without needing the Chrome Web Store.

### 1. Download the extension

1. Go to the [Releases](https://github.com/ba1234scx234s/ag-styler/releases) page
   *(or click the green **Code** button → **Download ZIP** to get the latest code)*
2. Unzip the downloaded file to a folder on your computer

### 2. Load it in Chrome

1. Open Chrome and go to **`chrome://extensions`**
2. Enable **Developer mode** using the toggle in the top-right corner
3. Click **Load unpacked**
4. Select the unzipped `ag-styler` folder
5. The AG Styler extension will appear in your extensions list

### 3. Pin it to your toolbar *(optional)*

Click the puzzle piece icon (🧩) in the Chrome toolbar and click the pin icon next to AG Styler so it's always visible.

---

## Usage

1. Navigate to any page on **autogate.co**
2. Click the **AG Styler** icon in your toolbar
3. Toggle **font swap** on or off
4. Add one or more **colour swaps** — enter the old hex and the new hex (with or without the `#`)
5. Click **Apply**

To undo all changes, click **Reset**.

---

## Updating

When a new version is available:

1. Download and unzip the latest code (same as step 1 above), replacing the old folder
2. Go to `chrome://extensions` and click the **reload** button (↺) on the AG Styler card
